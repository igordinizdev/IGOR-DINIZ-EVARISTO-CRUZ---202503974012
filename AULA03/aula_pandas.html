Conceitos Fundamentais

Os tijolos do Pandas: Series, DataFrame e primeiros passos

Estrutura de Dados

📊 DataFrame

Uma tabela completa com linhas e colunas — como uma planilha inteligente. É a estrutura principal do Pandas.

Pense como uma tabela SQL ou uma aba do Excel, mas com superpoderes de programação.

Estrutura de Dados

📈 Series

Uma única coluna de dados com índice — unidimensional. É o bloco básico que compõe o DataFrame.

Pense como uma lista com rótulos — cada elemento tem um índice associado.

Análise do Código

Cada operação explicada com exemplos e contexto

1
📁 Leitura e Escrita de Arquivos
read_csv() · to_csv() · to_excel()
▼

A porta de entrada de qualquer análise. O Pandas suporta CSV, Excel, JSON, SQL e dezenas de outros formatos.

01_leitura.py 
# Ler CSV padrão (vírgula)
df = pd.read_csv("dados.csv")

# CSV com separador ponto-e-vírgula (comum no Brasil)
df = pd.read_csv("livros.csv", sep=";")

# Salvar como CSV (index=False evita coluna extra)
df.to_csv("saida.csv", index=False)

# Exportar para Excel (requer pip install openpyxl)
df.to_excel("resultado.xlsx", index=False)

💡 Dica: Sempre use index=False ao salvar. Sem ele, o Pandas adiciona uma coluna numérica extra desnecessária.

2
🧹 Tratamento de Dados Faltantes
fillna() · dropna() · isnull()
▼

Dados reais quase sempre têm valores ausentes (NaN). Tratar corretamente é fundamental para não distorcer análises.

02_nulos.py 
# Verificar nulos em cada coluna
print(df.isnull().sum())

# Preencher com média (bom para dados sem outliers)
df["idade"] = df["idade"].fillna(df["idade"].mean())

# Preencher com mediana (mais robusta a outliers)
df["salario"] = df["salario"].fillna(df["salario"].median())

# Remover linhas com nulos
df_limpo = df.dropna()

# Remover nulos apenas em coluna específica
df_limpo = df.dropna(subset=["salario"])

💡 Regra: Use média quando os dados são simétricos. Use mediana quando há valores extremos (outliers) que distorceriam a média.

3
🔍 Filtragem e Ordenação
operadores booleanos · sort_values()
▼

Selecione apenas os registros relevantes com condições lógicas, igual ao WHERE do SQL.

03_filtros.py 
# Filtro simples por valor
filtro = df[df["salario"] > 3500]

# Múltiplas condições: & = AND, | = OR
filtro2 = df[(df["salario"] > 3000) & (df["cidade"] == "SP")]

# Filtrar lista de valores (como IN do SQL)
cidades = df[df["cidade"].isin(["SP", "RJ"])]

# Ordenar por salário (descendente)
df_ord = df.sort_values(by="salario", ascending=False)

# Ordenar por múltiplas colunas
df_multi = df.sort_values(by=["cidade", "salario"], ascending=[True, False])
4
📊 GroupBy: Agregações Poderosas
groupby() · agg() · pivot_table()
▼

O GroupBy é a operação mais analítica do Pandas — equivalente ao GROUP BY do SQL. Agrupa registros e aplica funções de resumo.

04_groupby.py 
# Média de salário por cidade
media = df.groupby("cidade")["salario"].mean()

# Múltiplas agregações ao mesmo tempo
resumo = df.groupby("cidade")["salario"].agg([
    "mean", "min", "max", "count"
])

# GroupBy com reset_index (volta a ser DataFrame normal)
resultado = (df.groupby("cidade")["salario"]
               .mean()
               .reset_index()
               .sort_values("salario", ascending=False))

⚡ Apply: Funções Customizadas
apply() · lambda · funções personalizadas
▼

O apply() permite criar novas colunas com qualquer lógica customizada usando funções Python normais ou lambdas.

05_apply.py 
# Criar coluna calculada
df["salario_anual"] = df["salario"] * 12

# Categorizar com lambda (função anônima)
df["categoria"] = df["salario"].apply(
    lambda x: "Alto" if x > 4500 else "Médio" if x > 3000 else "Baixo"
)

# Função mais complexa
def classificar_paginas(n):
    if n < 150:
        return "Curto"
    elif n <= 350:
        return "Médio"
    else:
        return "Longo"

df["faixa_paginas"] = df["paginas"].apply(classificar_paginas)
6
🔗 Merge: Combinando DataFrames
merge() · concat() · tipos de join
▼

O merge() combina dois DataFrames como um JOIN do SQL. Essencial para trabalhar com múltiplas tabelas ou fontes de dados.

06_merge.py 
# INNER JOIN: apenas correspondências dos dois lados
df_merge = pd.merge(df, bonus, on="nome")

# LEFT JOIN: todos do df, mesmo sem correspondência
df_left = pd.merge(df, bonus, on="nome", how="left")

# RIGHT JOIN e OUTER JOIN
df_right = pd.merge(df, bonus, on="nome", how="right")
df_outer = pd.merge(df, bonus, on="nome", how="outer")

# Junção com nomes de colunas diferentes
pd.merge(df1, df2, left_on="id_cliente", right_on="cliente_id")

template_exercicio.py 
import pandas as pd
import numpy as np

# ── 1. CARREGAR O DATASET ──────────────────────────────
df = pd.read_csv("livros.csv", sep=";")
print(f"Shape: {df.shape}")  # (linhas, colunas)

# ── 2. EXPLORAÇÃO INICIAL ─────────────────────────────
print(df.head())
print(df.info())
print(df.describe())
print(df.isnull().sum())

# ── 3. LIMPEZA ────────────────────────────────────────
# Preencher nulos de 'ano' com a mediana
mediana_ano = df["ano"].median()
df["ano"] = df["ano"].fillna(mediana_ano).astype(int)

# Remover livros com 0 páginas
df_limpo = df[df["paginas"] > 0].copy()

# ── 4. TRANSFORMAÇÃO ──────────────────────────────────
def faixa_paginas(n):
    if n < 150: return "Curto"
    elif n <= 350: return "Médio"
    else: return "Longo"

df_limpo["faixa_paginas"] = df_limpo["paginas"].apply(faixa_paginas)
df_limpo["decada"] = (df_limpo["ano"] // 10) * 10

# ── 6. EXPORTAR ───────────────────────────────────────
df_limpo.to_excel("livros_analisados.xlsx", index=False)
